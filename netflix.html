Sobre o Projeto

HTML CSS JavaScript

A Página da Netflix foi desenvolvida durante a Imersão Front-End com IA da Alura, com o objetivo de aprender os fundamentos do desenvolvimento web utilizando HTML, CSS e JavaScript, além de explorar o uso de Inteligência Artificial como apoio no desenvolvimento de código.


Durante as aulas, foi criado um clone simplificado da interface da Netflix, no qual o usuário pode selecionar um perfil e acessar um catálogo de conteúdos. O projeto utiliza HTML semântico para estruturar a página, CSS para estilização e responsividade, e JavaScript para adicionar interatividade, manipulação do DOM e uso de Local Storage para salvar informações do usuário entre páginas.

Estruturação de páginas com HTML5

Código Fonte

index.js 
import { categories } from './data.js';
import { createCarousel } from './components/Carousel.js';

document.addEventListener('DOMContentLoaded', () => {
    const nomePerfil = localStorage.getItem('perfilAtivoNome');
    const imagemPerfil = localStorage.getItem('perfilAtivoImagem');

    if (nomePerfil && imagemPerfil) {
        const kidsLink = document.querySelector('.kids-link');
        const profileIcon = document.querySelector('.profile-icon');

        if (kidsLink) kidsLink.textContent = nomePerfil;
        if (profileIcon) profileIcon.src = imagemPerfil;
    }

    const container = document.getElementById('main-content');

    if (container) {
        categories.forEach(category => {
            const carousel = createCarousel(category);
            container.appendChild(carousel);
        });
    }
});
