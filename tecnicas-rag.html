Sobre o Projeto

Python LangChain Ollama Milvus RAG

O LangChain: Técnicas Avançadas de RAG foi desenvolvido como parte das atividades do curso Alura – LangChain: Técnicas Avançadas de RAG. O objetivo do projeto foi aprofundar o entendimento sobre arquiteturas de Retrieval-Augmented Generation (RAG), explorando técnicas para melhorar a recuperação de informações e a qualidade das respostas geradas por modelos de linguagem.


Durante o projeto, foi construído um pipeline capaz de carregar documentos em PDF, processá-los, gerar embeddings e armazená-los em um banco vetorial, permitindo que modelos de linguagem consultem essas informações para responder perguntas de forma mais contextualizada. O sistema utiliza LangChain para orquestração, Ollama para geração de embeddings e execução do LLM, e Milvus como banco de dados vetorial. Também foram exploradas técnicas avançadas de recuperação de contexto, como HyDE, para melhorar a busca semântica.

Conceitos Fundamentais Praticados

Carregamento e processamento de documentos com DirectoryLoader
Divisão de documentos em chunks com CharacterTextSplitter
Geração de embeddings semânticos para representar textos
Armazenamento e busca vetorial utilizando Milvus
Uso de modelos de linguagem locais com Ollama
Construção de pipelines com LangChain
Implementação de técnicas avançadas de recuperação, como HyDE
Estruturação de prompts e chains para geração de respostas
Integração entre LLM, embeddings e banco vetorial para sistemas RAG

Código Fonte

final.ipynb 
from langchain_community.document_loaders import DirectoryLoader
from langchain_text_splitters import CharacterTextSplitter
from langchain_ollama import OllamaEmbeddings, ChatOllama
from langchain_milvus import Milvus
from langchain.chains import RetrievalQA

# Carregamento dos documentos
loader = DirectoryLoader("./docs", glob="**/*.pdf")
documentos = loader.load()

# Divisão em chunks
splitter = CharacterTextSplitter(chunk_size=1000, chunk_overlap=200)
chunks = splitter.split_documents(documentos)

# Embeddings e banco vetorial
embeddings = OllamaEmbeddings(model="nomic-embed-text")
vectorstore = Milvus.from_documents(
    documents=chunks,
    embedding=embeddings,
    connection_args={"uri": "http://localhost:19530"}
)

# Retriever
retriever = vectorstore.as_retriever()

# LLM
llm = ChatOllama(model="llama3")

# Pipeline RAG
qa_chain = RetrievalQA.from_chain_type(
    llm=llm,
    retriever=retriever,
    return_source_documents=True
)

pergunta = "Quais são os principais conceitos explicados nos documentos?"
resposta = qa_chain({"query": pergunta})

print(resposta["result"])
